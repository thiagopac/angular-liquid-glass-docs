Liquid Glass Dropdown

An accordion-style dropdown built with two composable components. Each item independently controls its open state while sharing a single SVG distortion filter from the parent container.

Built with Angular signals and a single shared SVG filter for the entire dropdown tree.
  • Liquid Glass Card
  • Liquid Glass Button
  • Liquid Glass Dropdown
  • More coming soon...

Install via npm and import the components you need. No extra providers required.

Usage

Import both LiquidGlassDropdownComponent and LiquidGlassDropdownItemComponent. The container creates and manages the shared SVG filter; items inherit all visual settings via CSS custom properties.
