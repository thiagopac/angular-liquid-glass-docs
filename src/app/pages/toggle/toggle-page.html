Liquid Glass Toggle

A glass-surface switch with a sliding thumb and smooth spring transition. Supports Angular's [(checked)] two-way binding out of the box via model(). Use offColor and onColor to tint the track for each state.

Usage

Use [(checked)] for two-way binding. The label input is optional. Set onColor to any rgba to tint the track when active.
