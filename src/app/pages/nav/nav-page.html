Liquid Glass Nav

An iOS-style glass tab bar built with angular-liquid-glass-nav and angular-liquid-glass-nav-item. Active state is managed internally — just listen to (activeTabChange).

Usage

Wrap angular-liquid-glass-nav-item components inside angular-liquid-glass-nav. Each item projects an SVG icon and has a required label input. Active state is managed automatically.
